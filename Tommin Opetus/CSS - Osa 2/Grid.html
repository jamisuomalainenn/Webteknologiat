Sivun tyylit on Bootstrapin kautta

Jotta bootstrap saadaan toimimaan, on 'head'-osioon laitettava 'src'-tagiin pohja Getbootstrap.com sivustolta. Bootstrap tyylittelee html tiedoston automaattisesti. Kannattaa laittaa bootstrap projektiin ensitöikseen, koska se saattaa sotkea muotoiluja jälkikäteen laitettaessa.

Grid

CSS Grid layout on kaksiulotteinen taulukko, johon käyttöliittymän komponentit asetellaan. Kuten flexboxissa, gridissäkin on container, jonka sisää sisältö (“itemit”) asetellaan. Grid on todella sekava käyttää ja vaatiikin paljon aikaa sen opetteluun. Tämäkin verkkosivu on tehty käyttäen gridiä. En juuri osaa selittää mitä tein, kunhan tein. Ja tämmöinen tästä nyt sattui tulemaan! Semmoisen käsityksen sain, että Grid on pähkinän kuoressa kasa containereita. Maksimissaan 12 containeria, joita voi sitten haluamallaan tavalla ripotella HTML-sivulle. Tätäkään en osaa selittää miten toimii käytännössä. Kiitti ja kuitti! :D
